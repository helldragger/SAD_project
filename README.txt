Pour lancer le projet, lancez le jar avec java -jar SAD_project.jar

Les differents flags sont détaillés si vous utilisez un des arguments suivant: -h --help.
Par defaut la sortie UI par console de la simulation est desactivée, vous pouvez la reactiver avec le flag -v, ou --verbose.

Tout le reste est (presque) exhaustivement explicité dans le SAD report! Bonne lecture! :D
